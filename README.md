# Para rodar

## Instalações

Rode o seguinte comando:
```bash
yarn install
```
ou
```bash
npm install
```

## Desenvolvimento

Rode o seguinte comando:
```bash
yarn start
```
ou
```bash
npm start
```

## Produção

Rode o seguinte comando:
```bash
yarn build
```
ou
```bash
npm build
```
